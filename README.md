# MANIIRAM INDUSTRIES

A responsive corporate website developed using HTML, CSS, and JavaScript for MANIIRAM INDUSTRIES, a manufacturer and supplier of electricity distribution line materials based in Madurai, Tamil Nadu.

## Features

- Responsive Design
- Sticky Navigation Bar
- Hero Section
- Company Overview
- Products Showcase
- Manufacturing Process
- Why Choose Us
- Gallery
- Contact Form
- Mail Integration (mailto:)
- Google Maps Integration

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts (Poppins)
